# RFC 0022: Lesson Builder - 资产库 (Asset Library) 交互规范

**状态**: 🟡 草稿  
**作者**: AI Assistant  
**创建日期**: 2026-01-25  
**最后更新**: 2026-01-25

## 1. 摘要

本 RFC 定义了 **Lesson Builder 的资产库 (Asset Library)**。它作为课程零件的物理中心，负责资产的检索、管理以及将其引入任务流的交互规范。

## 2. 资产类型管理

资产库分类管理以下预先存在的实体：

- **Quizzes**：以 `.quiz` 结尾的测验脚本。
- **Media**：存放于存储服务器或 CDN 的视频 (Remote URL)、音频、图片。
- **Resources**：PDF 手册、词汇表。

## 3. 核心交互：拖拽引用 (Drag-and-Refer)

资产库与流程画布 (RFC 0018) 的联动逻辑：

- **拖入新增**：从库中拖出资产 -> 在画布释放 -> **自动创建一个对应类型的 Node**。
- **绑定现有节点**：将资产拖到已有的空白 Node 上 -> 执行 **Asset Binding**。
- **引用而非拷贝**：所有的操作仅针对资产的 URI。

## 4. UI 行为

- **实时检索**：对海量 Quiz 资产提供按级别 (Level)、主题 (Theme) 的搜索过滤。
- **快速概览**：在库中悬停资产时，显示其元数据（题目数、视频时长、作者）。
